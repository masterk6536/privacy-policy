개인정보 처리방침

본 애플리케이션(이하 "앱")은 사용자로부터 별도의 개인정보를 수집하지 않으며, 다음과 같은 방식으로 정보를 처리합니다.

1. 수집하는 정보

  • 전화 수신 상태 감지 (READ_PHONE_STATE)
  • 전화 자동 거절 (ANSWER_PHONE_CALLS, MANAGE_OWN_CALLS)
  • 문자 전송 (SEND_SMS)
  • 부팅 시 자동 실행 (RECEIVE_BOOT_COMPLETED)
  • 알림 표시 (POST_NOTIFICATIONS)

2. 정보의 이용 목적

  • 사용자가 설정한 시간대에 전화가 오면 자동으로 거절
  • 지정한 메시지를 발신자에게 자동 전송
  • ON 상태 유지 설정 값 관리

3. 정보 보관 및 삭제

앱은 어떠한 개인정보도 저장하거나 외부 서버에 전송하지 않으며, 모든 설정값은 사용자의 단말기 내에만 저장됩니다.

4. 외부 공유

앱은 수집된 정보를 외부와 공유하지 않으며, 제3자에게 제공하지 않습니다.

5. 이용자 권리

로그인이나 계정을 요구하지 않으며, 앱 삭제 시 모든 정보는 자동 삭제됩니다.
